El Caos de la concurrencia (Minimarket)
Aceite Botella

Sofia y Catalina compran en fracciones de segundo al mismo tiempo.

Milisegundo 1: Sofia compra 5 botellas. El backend le pregunta a la base de datos el stock y la base datos responde: 10 
Milisegundo 2: Catalina compra 3 botellas. El backend le pregunta a la base de datos el stock. Como Sofia aun no ha guardado, la base de datos responde 10.
Milisegundo 3: El backend procesa la compra de Sofia en memoria (10-5 = 5). le dice a la base de datos : "Guarda 5".
Milisegundo 4: El backend procesa la compra Catalina  en memoria (10-3 =7). le dice a la base de datos Guarda 7.

Resultado fatal: Vendiste 8 botellas reales, pero tu base de datos reporta que tienes 7 botelletas de stock restante.

Por que el proceso de la compra de Catalina "aplasto" el trabajo del el proceso de compra de Sofia.
porque ambos procesos hicieron las matematicas sobre una "snapshot" (fotografia) estatica y desconectada de la base datos.
