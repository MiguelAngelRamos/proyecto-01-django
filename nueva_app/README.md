# Proyecto: Clínica Veterinaria (Django + Bootstrap 5)

Este directorio contiene la guía paso a paso para crear un proyecto completamente nuevo en Django con la temática de **Clínica Veterinaria**. 
Incluye un CRUD completo para gestionar los pacientes (mascotas).

## Índice de Guías

Por favor, sigue los siguientes archivos en orden para construir el proyecto:

1. **[01_configuracion_inicial.md](./01_configuracion_inicial.md)**: Creación del entorno, proyecto, aplicación y configuración de Base de Datos (`vet_clinic_db`).
2. **[02_modelos_y_admin.md](./02_modelos_y_admin.md)**: Creación del modelo `Mascota`, migraciones y registro en el panel de administración.
3. **[03_formularios.md](./03_formularios.md)**: Configuración de formularios de Django integrados con clases CSS de Bootstrap 5.
4. **[04_vistas_y_urls.md](./04_vistas_y_urls.md)**: Creación del CRUD utilizando Vistas Genéricas (CBVs) y enrutamiento (URLs).
5. **[05_templates_base_y_crud.md](./05_templates_base_y_crud.md)**: Plantillas HTML utilizando CDN de Bootstrap 5 para el Navbar, Tablas y Formularios.

¡Sigue paso a paso cada documento y tendrás tu nueva aplicación funcionando!
